WriterEvent Type

Namespace: Fantomas.Core
Assembly: Fantomas.Core.dll
Base Type: ValueType
All Interfaces: IEquatable<WriterEvent> , IStructuralEquatable , IComparable<WriterEvent> , IComparable , IStructuralComparable
Kind: Struct

Represents a single event emitted during the code formatting process. The sequence of writer events captures how the formatter produces its output.
