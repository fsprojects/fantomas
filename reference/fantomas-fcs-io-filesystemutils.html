Full Usage: checkSuffix path suffix

Parameters:

Returns: bool

checkSuffix f s returns True if file name "f" ends in suffix "s", e.g. checkSuffix "abc.fs" ".fs" returns true. Disregards casing, e.g. checkSuffix "abc.Fs" ".fs" returns true.
